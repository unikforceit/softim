== Installation ==
Manual installation:

1. Upload the `softim` folder to the `/wp-content/themes/` directory

Activation and Use

1. Activate the Theme through the 'Themes' menu in WordPress
2. See Theme Settings to change theme specific options

== Custom Logo ==
This theme supports custom logo for the header part. 
You can change or remove it from Theme Option -> Header.
